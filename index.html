Get it

Ok here's how to install it.

It's a Chrome extension. If you've never installed one outside the Chrome Web Store before, don't worry — it sounds more technical than it is. I promise.

1 Download the ZIP file and unzip it. You'll get a folder with some files in it. Don't worry about what they are.
2 Open Chrome. Type chrome://extensions in the address bar and hit enter. It'll open a settings page.
3 In the top right corner, turn on Developer mode. It's just a toggle. This sounds scary but it's totally normal — it just means "let me install my own extensions."
4 Click Load unpacked (top left). Select the folder you unzipped. Done.
5 Go to therealreal.com. You'll see a dark toolbar at the bottom of the page. That's the extension. Start shopping.

The whole thing takes about 2 minutes. If you get stuck, text me.

The backstory

I'd never written a line of code before this.

I'm not a developer. I just shop on The RealReal a lot, and every week I'd run into the same annoying problems — re-entering my filters over and over, losing track of things I wanted, scrolling past a wall of sold items to find anything available.

At some point I thought, why doesn't The RealReal just... fix this? And then I thought, ok, what if I just fix it myself?

So I coded it with Claude, learned how Chrome extensions work, broke a lot of stuff, and eventually got it working. This is literally my first coding project ever. It's on GitHub if you want to look at the code, suggest features, or tell me it's broken.
