# RealReal Enhancer ⚡

A Chrome extension that makes shopping [The RealReal](https://www.therealreal.com) way faster.

If you've ever spent an hour re-applying the same filters across different designers, losing items to "SOLD" while you're still browsing, or scrolling through pages of stuff you can't buy — this fixes that.

---

## What it does

### Filter Presets
Save your filters once (size, condition, sort order, etc.) and apply them to any designer or search page with one click. No more re-setting everything every time you switch from Ganni to Rag & Bone.

### Item Labels
Hover over any product in the grid to tag it with a label — 🛒 Want to Buy, 📌 Reference, 🎁 For a Gift, or your own custom labels. Colored dots show on items you've already tagged so you can see them at a glance while browsing.

### Compact View
Squeezes more items per row so you can scan faster.

### Dim Sold Items
Fades out sold items so they're not an eyesore cluttering up the grid. You can still see them — which is actually useful for spotting styles you like and searching for something similar — but they're not competing for your attention with items you can actually buy.

## Install (2 minutes)

1. Click the green **Code** button above → **Download ZIP**
2. Unzip the downloaded file
3. Open Chrome and go to `chrome://extensions`
4. Turn on **Developer mode** (toggle in the top-right corner)
5. Click **Load unpacked** and select the unzipped folder
6. Go to [therealreal.com](https://www.therealreal.com) — you'll see a toolbar at the bottom of the page

---

## How to use

### Saving a filter preset
1. Browse The RealReal and apply your filters normally (designer, size, condition, sort, etc.)
2. Click **+ Save Current Filters** in the bottom toolbar
3. Name it, pick an emoji, and save
4. Now switch to a different designer and click your preset — same filters, new page

### Tagging items
1. Hover over any product card in the grid
2. Click one of the emoji buttons that appear (🛒 📌 🎁)
3. Colored dots will show on tagged items as you browse
4. View all your tagged items in the extension popup (click the ⚡ icon)

### Managing everything
- **Hover** over a preset for ✎ edit or ✕ delete
- **Shift+Click** a preset to go to the exact original page instead of applying filters
- **Right-click** a preset in the toolbar to delete it
- Click the **⚡ icon** in your Chrome toolbar for the full dashboard (presets, saved items, settings)
- Add custom labels in Settings → Item Labels

## Tips

- **Create presets for shopping modes, not designers.** A preset called "Basics" (size M, Like New, price low→high) works across every designer. One called "Shoe Hunt" (size 7.5, shoes only) works across every brand.
- **Tag as you browse.** It's faster to tag 20 things in 10 minutes and then decide what to buy, than to click into each item one at a time.
- **Use Compact View** when you're scanning. Turn it off when you want to see details.

---

## Troubleshooting

**Toolbar doesn't appear:** Make sure you're on therealreal.com and the extension is enabled in `chrome://extensions`. Try refreshing the page.

**Compact view looks off:** The RealReal updates their site layout periodically. The compact view uses CSS overrides that may need tweaking — open an issue and we can update them.

**Presets not applying filters:** Some pages handle filters differently. If a preset doesn't work on a particular page, try Shift+Click to go to the original saved page instead.
